## Como rodar localmente

- Clone o repositório:
```
git clone https://github.com/RaffaellaSantos/portfolio.git  
cd portfolio  
```

- Suba os serviços com Docker Compose:
```
docker-compose up --build -d
```

Verifique docker-compose.yml para detalhes das urls.

## Hospedagem

- Banco de dados: Aiven

- Backend: Render

- Frontend: Vercel
